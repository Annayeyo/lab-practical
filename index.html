USER LOGIN PAGE
Lab Practical -3
  1. Objective: Create a simple login page using HTML
  2. Theory:The form element is used to create the form, and the action attribute which tells the browser where to send the form data when the user submits it. The method attribut is set to "post" which tells the browser to send the data in the body of the HTTP request.

    The label element is used to create a text label for the input fields, and the for attributes is used to associate the label with corresponding input fields. The input elements are used to create the text fields for the username and password. The type attribute is set to "text" for the username field and "password" for the password field, which tells the browser to mask the characters entered in the password field.

    The required attribute is added to the input fields, which tells the browser that these fields must be filled out before the form can be submitted.

    The input tupe="submit" element is used to create a submit button, and the value attribute is set to "login" which will be displayed on the button.
